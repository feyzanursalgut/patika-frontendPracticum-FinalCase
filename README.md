patika-frontendPracticum-FinalCase

You can run it in terminal for a start.

git clone https://github.com/feyzanursalgut/patika-frontendPracticum-FinalCase.git

After the project file is opened, run the following command.

npm i 

After completing the download, type the following command in the terminal.

npm start

If you haven't received any errors up to this point, http://localhost:3000/ is ready for you.
